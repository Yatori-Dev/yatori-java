# Yatori-core

---

## 介绍：

> Hello，这里是Yatori-core。Yatori-core是所有Yatori衍生产品的核心，所有Yatori衍生产品都将会通过Yatori-core提供的对外开放接口进行开发。

## 英华在线
* 获取课程
  * 课程信息
    * 获取课程名称
  * 课程视屏
    * 单视屏操作（获取视屏总时长，获取当前视屏学时，开始学习该视屏，暂停学习该视屏）
  * 课程考试
    * 单考试操作
